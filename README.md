# Flask Rental System 房屋租赁系统

本项目是一个基于 **Flask** 框架开发的房屋租赁综合管理系统，包含**商家端**、**管理端**、**客户端**三大模块。系统支持房源管理、租赁流程、数据分析、智能推荐、房价走势预测等丰富功能，适用于租客、房东/商家和平台管理员不同角色的多场景需求。

---

## 系统架构

- **客户端（租客端）**：租客浏览、搜索、收藏、预约房源，签约管理等。
- **商家端（房东端）**：房源发布、编辑、上下架、订单管理、合同管理等。
- **管理端（平台后台）**：用户管理、房源审核、数据统计与监控、权限分配等。

---

## 主要功能列表

### 一、客户端（租客端）
- 首页推荐、热门房源、最新房源展示
- 多条件搜索（区域/小区/户型/价格/面积/关键字）
- 房源详细信息展示（图片、户型、面积、配套设施等）
- 收藏/取消收藏房源
- 在线预约看房（预约状态跟踪）
- 智能推荐房源（基于用户行为与兴趣）
- 查看同小区/相似房源推荐
- 房价走势分析与AI预测（历史房价、未来走势、户型价格对比等）
- 数据可视化（户型占比、价格分布、热门小区等）
- 在线签约、合同下载
- 历史浏览记录
- 个人中心（信息管理、预约管理、收藏管理、合同管理等）
- 登录、注册、找回密码

### 二、商家端（房东端）
- 房源发布（多图上传、详细配置）
- 房源编辑、下架、删除
- 管理已发布房源（查询、筛选、统计）
- 预约管理（查看租客预约、确认/拒绝预约）
- 租赁合同管理（签署、查看、下载）
- 房源数据分析（浏览量、收藏量、成交量等）
- 房源状态实时监控
- 个人信息及账户管理
- 消息通知、平台公告查看

### 三、管理端（平台后台）
- 用户管理（租客/房东/管理员增删改查、权限分配）
- 房源审核与管理（违规房源下架、房源质量把控）
- 订单与合同管理
- 平台数据统计与报表（房源总数、交易量、活跃用户、收入等）
- 数据可视化大屏（实时监控房源分布、价格走势、预约情况等）
- 平台公告发布
- 登录记录与安全管理
- 权限管理与角色分配

## 数据分析与AI智能预测

- **房价走势预测**  
  基于历史房价数据与时间序列，利用 Sklearn 线性回归算法对未来价格进行预测，图表展示实际/预测价格，便于用户决策。

- **数据可视化**  
  基于 ECharts 实现多维度可视化：户型占比、房源数量TOP20、价格分布、房价趋势、热门小区等。

- **个性化推荐**  
  结合用户浏览/预约/收藏等行为，智能推荐相关房源，提高转化率。

## 技术栈

- **后端**：Flask、SQLAlchemy、Sklearn
- **前端**：HTML、CSS、Bootstrap、jQuery、ECharts
- **数据库**：MySQL
- **其他**：AJAX、RESTful API

## 部署与运行

1. 克隆项目
   ```bash
   git clone https://github.com/jasminelee162/flask_rental_system.git
   cd flask_rental_system
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 配置数据库（修改 `app.py` 或配置文件）

4. 初始化数据库（如有需要）

5. 启动服务
   ```bash
   python app.py
   ```

6. 浏览器访问 [http://localhost:5000](http://localhost:5000)
