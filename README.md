jasminelee162/flask_rental_system 项目实现了如下主要功能：

1. **首页功能**
   - 展示最新房源和最热房源（按发布时间和热度排序展示）。
   - 首页“为您推荐”展示精选房源。
   - 支持区域、商圈、小区名和户型搜索，搜索栏可切换。
   - 首页统计房源总数量。

2. **房源列表页**
   - 支持房源的分页浏览。
   - 可以根据发布时间（最新房源）或浏览量（最热房源）排序分页展示房源。
   - 搜索结果支持分页展示。

3. **房源详情页**
   - 展示房源详细信息，包括图片、地址、户型、面积、价格等。
   - 展示配套设施信息。
   - 支持房源收藏功能。
   - 展示同小区的其他房源推荐。
   - 展示户型占比（饼图）、小区房源数量TOP20（柱状图）、户型价格走势（折线图）、小区房价分布等可视化信息。

4. **数据可视化**
   - Echarts 图表展示：户型占比、区域小区房源数量TOP20、房价走势等。
   - 图表数据通过后端接口动态获取。

5. **辅助功能**
   - 搜索联想与智能提示。
   - 输入框中英文搜索适配。
   - 支持点击搜索结果自动填充输入框。
   - 简单的前端分页功能。

6. **后端接口与路由**
   - 提供用于数据查询和图表数据的多个API接口。
   - 实现房源的增删查改接口（部分可推断）。
